# 豆瓣电台1.0.0发布

打开豆瓣电台客户端，聆听动人的音乐，无需打开网页就能收听豆瓣电台。清新的界面，平滑的动画，快捷的操作，体验如此美妙。

运行环境：Windows XP, Vista, 7或更高版本的系统，需要.Net Framework 4.0、Windows Media Player 10或更高版本，要求网络连接。

1.0.0
